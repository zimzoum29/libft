# First Project of 42 School Common Core

# libft

## Usage

``make`` without bonus functions.

``make bonus`` to compile just bonuses.

## About Project

Libft is the first project of 42 School when you start the common core. 
The subject simply asks us to recode basic libc functions that we will have the opportunity to reuse in several school projects.
